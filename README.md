# Flipkart Laptop Data Analysis

## Project Overview
This project demonstrates an **end-to-end data analysis workflow**:
- Web scraping laptop data from an e-commerce website
- Data cleaning and preprocessing
- Exploratory data analysis (EDA)
- Visualization of insights

The objective is to analyze laptop pricing, ratings, and brand trends.

---

## Tech Stack
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- BeautifulSoup
- Requests
- Jupyter Notebook

---

## Project Workflow

### 1. Data Scraping
- Scraped laptop product data
- Extracted:
  - Product Name
  - Price
  - Rating
  - Reviews
  - Specifications


---

### 2. Data Cleaning
Performed:
- Handling missing values
- Formatting price columns
- Data type conversion
- Removing duplicates

---

## Key Insights
- Identified major laptop brands in the dataset
- Observed price distribution patterns
- Analyzed relationship between ratings and price

---

## How to Run the Project

1. Clone the repository
   git clone https://github.com/your-username/flipkart-laptop-data-analysis.git
